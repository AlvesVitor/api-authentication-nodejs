# Microserviço de autenticação com Nodejs

 O projeto consiste em um **microserviço de autenticação** e criação de usuários.

## Endpoints Base para consumir o serviço

### Usuários

* GET /users
* GET /users/:uuid
* POST /users
* PUT /users/:uuid
* DELETE /users/:uuid

### Autenticação

* POST /token
* POST /token/validate
